# AI Worklog

This folder is the persistent handoff and planning record for our Unity fixes.

- `TASKS.md`: active/pending/completed tasks
- `SESSION_LOG.md`: chronological record of what was changed
- `FUTURE_PLANS.md`: upcoming plans and follow-ups
- `TECH_DICTIONARY.md`: plain-English "how things work" reference (easy -> advanced)

Update rule:
- Every time we fix something, add a short entry.
- Every time we decide a plan, add/update it in `FUTURE_PLANS.md`.
- Every time behavior/flow changes, update `TECH_DICTIONARY.md`.
